**Congratulations!** You've completed the Mycelium Tutorial 🙌.

With fast lookup, flexible hierarchies, note linking, and rich formatting, you're now ready to organize, reference, and work with any amount of knowledge.

From here, you can start adding your notes to this workspace or, if you'd like to start fresh, you can create a new workspace for your notes with the `Mycelium: Initialize Workspace` Command.

## Publishing

Mycelium isn't just for taking notes. It can be used to share your thoughts and documentation with others. One of those ways is with **publishing**. The [Mycelium Documentation](https://wiki.mycelium.so) and [Mycelium Blog](https://blog.mycelium.so) are both published this way.

If you want to learn more about [publishing your notes online](https://wiki.mycelium.so/notes/e5st4LFLtIwwbQmC6JBaF), you can take a look at how to publish with example guides for [Netlify](https://wiki.mycelium.so/notes/yetuum6o9wZi6eVJQBbQb) or [GitHub Pages](https://wiki.mycelium.so/notes/yg3EL1x9fEe4NMqxUC3jP).

### Trail

_This is your immediate jumping off point after finishing the tutorial. It covers useful concepts that we didn't have time to cover in the basic tutorial._

- Basics
  - [x] Learn the basics of Mycelium with the [Getting Started tutorial](https://wiki.mycelium.so/notes/678c77d9-ef2c-4537-97b5-64556d6337f1).
  - [ ] Read up on the [Amoeba Workflow](https://wiki.mycelium.so/notes/e780000d-c784-4945-8e42-35218a3ecf10) - a highly effective way of taking notes in Mycelium
- Navigation
  - Learn how to [find all note references to a header](https://wiki.mycelium.so/notes/cphUwSPk12j4lS0tKjBdC).
  - Learn how to [find all broken, internal reference links](https://wiki.mycelium.so/notes/ZeC74FYVECsf9bpyngVMU).
  - Learn how to navigate between notes using [navigation commands](https://wiki.mycelium.so/notes/cphUwSPk12j4lS0tKjBdC).
- Organizing
  - Learn to expand your organization beyond hierarchies by [using tags](https://wiki.mycelium.so/notes/8bc9b3f1-8508-4d3a-a2de-be9f12ef1821).
  - Learn how to [tag other Users](https://wiki.mycelium.so/notes/ESqOqSOEo54lqvGLlQXM6) in your notes.
  - Learn how to [manage and track tasks with special Task Notes](https://wiki.mycelium.so/notes/SEASewZSteDK7ry1AshNG).
  - Learn how to quickly take notes that may note have a home quite yet, like a scratchpad, using [Scratch Notes](https://wiki.mycelium.so/notes/5c213aa6-e4ba-49e8-85c5-1bdcb33ce202).
  - Learn how to log your activities with [Daily Journal Notes](https://wiki.mycelium.so/notes/ogIUqY5VDCJP28G3cAJhd) and other generic [Journal Notes](https://wiki.mycelium.so/notes/5c213aa6-e4ba-49e8-85c5-1bdcb33ce202).
  - Learn the _how_ and the _why_ behind [Hierarchies in Mycelium](https://wiki.mycelium.so/notes/f3a41725-c5e5-4851-a6ed-5f541054d409).
  - Learn how to automatically make use of [Templates](https://wiki.mycelium.so/notes/861cbdf8-102e-4633-9933-1f3d74df53d2) in new notes, in specific hierarchies, with [Schemas](https://wiki.mycelium.so/notes/c5e5adde-5459-409b-b34d-a0d75cbb1052).
- Sharing
  - Learn how to backup your vaults online, privately or accessible to the public, with [git and GitHub](https://wiki.mycelium.so/notes/vsAxYld4zsbD2XjKoh7OP).
  - Learn about popular VS Code extensions, helpful tools, and other resources from the Mycelium community from [`awesome-mycelium`](https://github.com/myceliumhq/awesome-mycelium/).

### Forest

_Here we go a level deeper by going over intermediate topics. Wander through here to learn how to customize Mycelium for your workflow._

- Retrieval
  - Learn how to customize the power and functionality of your lookups with [Lookup Modifiers](https://wiki.mycelium.so/notes/ad270a7d-2aed-4273-8319-eb6536e38b29).
- Editing
  - Learn how to understand and extend the metadata of your notes with [Frontmatter Attributes](https://wiki.mycelium.so/notes/ffec2853-c0e0-4165-a368-339db12c8e4b).
- Organizing
  - Learn how to [manage and understand different kinds of Vaults](https://wiki.mycelium.so/notes/6682fca0-65ed-402c-8634-94cd51463cc4).
  - Learn how to [manage and understand different kinds of Workspaces](https://wiki.mycelium.so/notes/c4cf5519-f7c2-4a23-b93b-1c9a02880f6b).
  - Learn how to work with [Multiple Vaults in the same workspace](https://wiki.mycelium.so/notes/24b176f1-685d-44e1-a1b0-1704b1a92ca0). This allows you to access private personal notes from a vault, while working with notes from other vaults (like team notes, or software documentation in a project repository).
- Sharing
  - Learn how to [publish your notes online](https://wiki.mycelium.so/notes/e5st4LFLtIwwbQmC6JBaF), and follow guides for hosting with [Netlify](https://wiki.mycelium.so/notes/yetuum6o9wZi6eVJQBbQb) or [GitHub Pages](https://wiki.mycelium.so/notes/yg3EL1x9fEe4NMqxUC3jP).
  - Learn how to [access your Mycelium notes on mobile devices with third-party software](https://wiki.mycelium.so/notes/401c5889-20ae-4b3a-8468-269def4b4865).
- Transfer
  - Learn how to use Mycelium interoperably with other formats, by [exporting/importing your notes with Pods](https://wiki.mycelium.so/notes/66727a39-d0a7-449b-a10d-f6c438185d7f).
